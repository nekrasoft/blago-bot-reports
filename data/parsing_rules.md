# Правила парсинга для бота

## Азбука

При упоминании адреса "козулева" (в любом варианте: козулева 2/4, козулева 9к1 и т.д.)  
**Примечание** всегда = "знак".

## Железно / Железно-Киров

- "железно зарядное" → Контрагент = Железно, Примечание = Зарядное
- "железно знак" / "железно инноград" → Контрагент = Железно-Киров, Примечание = знак или инноград

## Управление домами

- "жуковского" / "жуковского 7" → Примечание = Жуковского
- "свободы" / "свободы 121а" → Примечание = Свободы

## Ходка (вывоз с земли)

- "ходка" / "ходки" + название клиента → тип trip_removal, Объект = кол-во ходок (по умолчанию 1)
- "акмаш" → Акмаш-Холдинг или Акмаш
- "маяк" → Маяк (добавить в справочник при необходимости)
